# ULE

## Usuarios Libres de Esquel

ULE es una ONG que busca promover el software y hardware libre en la localidad de Esquel, Chubut, Argentina.
Fue creado hace más de 10 años por estudiantes, profesores y vecinos de la Escuela N°713 "Juan Abdala Chayep" y fue originalmente una de las primeras escuelas en ser capaz de certificar el uso e instalación de software libre por la Organización Latinux.
